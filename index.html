실제 조명기 정보 추가됨

체이스 편집 (CHASE EDIT)

  1. BLACK 버튼으로 정지 모드 진입
  2. EDIT/RUN 버튼으로 편집 모드 전환
  3. P1-P12 버튼으로 체이스 번호 선택
  4. P1-P12 버튼으로 조명기 선택
  5. 슬라이더로 각 채널 값 설정
  6. SPEED/CROSS 노브로 타이밍 설정
  7. + 버튼으로 현재 씬을 체이스에 저장
  8. 추가 씬이 필요하면 4-7단계 반복
  9. EDIT/RUN 버튼으로 편집 완료

체이스 실행 (CHASE RUN)

  1. EDIT/RUN 버튼으로 실행 모드 확인
  2. P1-P12 버튼으로 실행할 체이스 선택
  3. SPEED 노브로 체이스 속도 조절
  4. CROSS 노브로 페이드 타이밍 조절

수동 제어 (MANUAL 모드)

  1. MANUAL 버튼 누르기
  2. P1-P12로 제어할 조명기 선택
  3. 해당 조명기 슬라이더로 각 채널 조작
  4. RELEASE CHANNELS로 자동 제어 복귀

P1 조명기 콘솔 매핑

CH1 → P1 밝기 (채널 1)
CH2 → P1 깜빡임 (채널 2)
CH3 → 미사용
CH4 → P1 프리셋 컬러 (채널 4)
CH5 → P1 빨강 (채널 5)
CH6 → P1 녹색 (채널 6)
CH7 → P1 파랑 (채널 7)

RUN SCENE 모드 주의사항

중요: RUN SCENE 모드에서는 +/- 버튼으로 씬 간 전환이 불가능합니다. 이는 콘솔의 기본 제한사항입니다.

체이스 실행 중 수동 제어

P1 조명기도 체이스 실행 중 수동 제어 가능:

  1. MANUAL 버튼 누르기
  2. P1 버튼으로 P1 조명기 선택
  3. 원하는 채널 슬라이더 조작
  4. RELEASE CHANNELS로 자동 모드 복귀

LCD 표시 한계

현상: 채널별 정확한 DMX 값이 LCD에 표시되지 않음

대안: 이 앱의 가상 컨트롤 패널로 값 확인

RUN SCENE 제한사항

현상: RUN SCENE에서 +/- 버튼이 작동하지 않음

해결: P1 조명기는 CHASE 모드 사용 권장

P1 조명기 설정 문제

체크리스트:

  • P1 조명기가 DMX 1번 주소로 설정되어 있는지 확인
  • DMX 케이블이 올바르게 연결되어 있는지 확인
  • 콘솔의 P1 버튼이 P1 조명기와 매핑되어 있는지 확인
  • 각 채널 슬라이더가 올바른 기능에 연결되어 있는지 확인

DMX 연결 트러블슈팅

1단계: 물리적 연결

• DMX OUT → P1 조명기 IN

• 케이블 상태 확인

2단계: 주소 설정

• P1 조명기 = DMX 1-7번

• 조명기 뒷면 DIP 스위치 확인

3단계: 신호 테스트

• 채널 1 (밝기) 슬라이더로 반응 확인

• RGB 채널로 색상 변화 확인
